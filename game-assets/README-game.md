# Dừa & Mixue: Chạy Trốn Sói! 🐺🦖

## Đưa vào repo đã có sẵn (không đè lên index.html)

1. Copy vào repo:
   - `game.html`
   - thư mục `game-assets/` (gồm `dua.jpg`, `mixue.jpg`)
2. Đặt `bg-music.mp3` cùng cấp với `game.html` (game gọi đường dẫn tương đối `bg-music.mp3`).
3. Cấu trúc:
   ```
   your-repo/
     index.html          (trang chính có sẵn của bro)
     game.html            (game mới)
     bg-music.mp3
     game-assets/
       dua.jpg
       mixue.jpg
   ```
4. Push lên GitHub, truy cập `yourdomain.github.io/game.html` để chơi.
5. Muốn link từ trang chính sang game: thêm 1 nút/link `<a href="game.html">Chơi game</a>` trong `index.html`.

## Cách chơi
- Chọn 1 trong 4 nhân vật: Dừa, Mixue, Công chúa, Hoàng Tử
- Chọn độ khó: Dễ / Trung bình / Khó
- Nhấn phím CÁCH (hoặc chạm màn hình) để nhảy né sói Jack, sói Hector, khủng long
