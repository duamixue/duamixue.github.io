Thêm bài học mới

Mở file items.json (KHÔNG cần sửa index.html) và thêm một object vào mảng:

{ "title": "Tên bài học", "file": "ten-file.html", "emoji": "🎯", "subject": "Toán", "type": "Bài học", "grade": "Lớp 1", "gradeLevel":[1], "desc": "Mô tả ngắn.", "date": "2026", "color": "linear-gradient(135deg,#1a1a2a,#0d0d18)", "featured": false, "essential": false }

Lưu ý: subject nhận Toán | Tiếng Việt | Tiếng Anh | Khoa học | Tư duy | Nghệ thuật | Giải trí... · type nhận Bài học | Luyện tập | Trò chơi | Khám phá | Thực hành · gradeLevel là mảng số lớp (VD [1,2]), để [] nếu "Mọi lứa tuổi" — dùng để tự lọc theo độ tuổi (Lớp 1→6 tuổi, Lớp 2→7 tuổi...) · JSON bắt buộc dùng dấu ngoặc kép "key", không dùng comment //, không để dấu phẩy thừa ở object cuối.

  • 2

  • 4
    Vào Settings → Pages, chọn branch main → Save
  • 5
    Truy cập ngay tại tenban.github.io/my-portal 🎉
